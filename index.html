Answers 1

1. What components are part of the Von Neuman Architecture?

CPU, memory and input/output

2. What is user space?

All the code that runs outside of the operating system's kernel.

3. What is the shell?

The shell is a tool for manipulating data and executing several applications.

4. Where is the shell located?

The shell is located in the user space.

5. What is the main task of the shell?

The main task of the shell is to read commands and run other programs.

22. Script: Store the output of the command "hostname" in a variable. Display "This script is running on_." where "_" is the output of the "hostname" command.

23. Script: Store the output of the working directory (from where the script is run) in a variable then print the variable
